# Fleximy - Plataforma de Gestión de Contenido Creativo

## Descripción
Fleximy es una plataforma integral para la gestión de contenido creativo que ofrece servicios de diseño gráfico, branding, motion graphics, UX/UI y más. Sin contratos, precios asequibles y entregas rápidas.

## Tecnologías Utilizadas
- **Frontend**: React 18 + Vite
- **Animaciones**: GSAP (GreenSock)
- **Estilos**: CSS3 con variables CSS
- **Fuentes**: Plus Jakarta Sans, DM Serif Display
- **Deployment**: Netlify
